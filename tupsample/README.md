# Java21

These are some of the curated examples from different Github repos. The intention was to showcase new Java language 
features to TUP Manila faculties of Computer Engineering.

### Features

Records, Pattern Matching for instanceof, Text Blocks (multiline strings), Emoji, Sequenced collections.

### Requirements:

Java 21
- [JDK 21](https://jdk.java.net/21/)
- [Alternative | Eclipse Temurin](https://adoptium.net/)
